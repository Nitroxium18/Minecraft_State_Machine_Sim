# Minecraft State Machine Sim

A terminal based survival simulator inspired by Minecraft style state changes

## Overview

This project models a simple survival game using a state machine. The player has changing stats such as health, hunger, time and inventory. Actions update the game state and move the simulation forward

## Features

- Terminal text input based gameplay
- Player state tracking
- Basic action system
- Time progression
- Simple inventory structure

## How to Run

After you clone or download the repository, open a terminal in the project root and run:

python src/main.py
